# Expected Outputs — Online Day 3 PM

## Sensitivity-analysis plan

- [ ] Names one specific structural parameter (not a data input) and its baseline value, cited or marked `[SME REVIEW NEEDED]`.
- [ ] States a specific perturbation (value or % change), not "we would change it a bit."
- [ ] States an expected effect with a mechanism (which output indicators, which direction, why), not just "GDP would change."
- [ ] Lists the accounting-coherence checks that would need to pass before trusting the perturbed result.
- [ ] Only one parameter is perturbed — not a combination.
- [ ] Any uncertain file location is marked `[SME REVIEW NEEDED]`, not guessed.
