Finance & energy-efficiency scenarios, reserve requirements, and calibration & sensitivity analysis — continuing your DGE‑METRIC training online, in depth.

Who it's for

Graduates of the on-site DGE-METRIC training. This module doesn't repeat modeling theory — it picks up where the classroom left off and moves straight into applied, independent model use: calibration, scenario design, interpretation, and policy communication.

What you'll leave able to do

  • Navigate the DGE-METRIC repository, down to tracing an equation to its code
  • Run a baseline calibration, debug common errors, and read diagnostics with confidence
  • Fully document a finance scenario and an energy-efficiency scenario, with transparent assumptions
  • Modify calibration inputs and structural parameters, and run a sensitivity analysis
  • Explain the optimal-reserve-requirement method (MC = MB) and how it would be implemented
  • Use GitHub Desktop and VS Code, plus an AI coding assistant, for your own model work
  • Produce reproducible charts and honest policy narratives — caveats included
