## Visual Studio Code + GitHub Copilot Setup

Professional participant guide (15 July 2026).

## 1. Prerequisites

- Windows access with permission to install applications/extensions
- Stable internet connection
- GitHub account with Copilot trial or paid access

## 2. Install Visual Studio Code

1. Download VS Code from https://code.visualstudio.com
2. Run the installer and complete setup.
3. Open Visual Studio Code.

## 3. Enable GitHub Copilot in VS Code

1. Open Extensions view (`Ctrl+Shift+X`).
2. Install the GitHub Copilot extension.
3. Install the GitHub Copilot Chat extension.
4. Sign in with your GitHub account when prompted.
5. Authorize VS Code in your browser and return to the editor.

## 4. Verify It Works

- Open any code file and begin typing.
- Confirm inline suggestions appear.
- Press `Tab` to accept a suggestion.
- Open Copilot Chat and ask a sample coding question.

## 5. Helpful Links

- Getting started docs: https://docs.github.com/copilot
- Check license status: https://github.com/settings/copilot

Tip: If extensions are blocked by your organization, ask IT to allow GitHub and VS Code Marketplace access.
